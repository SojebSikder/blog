# blog
Blog app created with laravel and reactjs

## Installing
```
yarn install
```
## Setup
- Copy .env.copy to .env And set up database credentials in env file
- run this command for database migration: `php artisan migrate`

## Running
First compile react js app:
```
yarn prod
```
Run laravel app:
```
php artisan serve
```

## For development
Run this command:
```
yarn watch
```
## Technology used
- Laravel
- React js
- Tailwind css
